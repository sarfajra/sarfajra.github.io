About Website

Ini adalah website pertama yang saya buat untuk menguji kemampuan saya dalam membuat dan mengelola sebuah website. Proses pembuatan website ini membutuhkan waktu yang cukup lama karena saya memulainya dengan membuat diagram alir (flow chart), kemudian membuat sketsa desain, desain tampilan halaman menggunakan Canva, dan mengetik semua kode yang menghasilkan website ini.

Saya mendapatkan banyak pelajaran dari pembuatan website ini, seperti mengatasi masalah sendiri ketika menemui kesalahan, merancang tampilan website sesuai dengan keinginan, meningkatkan pemahaman tentang bahasa pemrograman yang digunakan, melatih kesabaran, memahami script-script, belajar mengelola database, dan masih banyak lagi.

Website ini juga dibuat sebagai latihan untuk menggabungkan beberapa proyek yang telah saya buat selama belajar coding, mulai dari form login dan sign up, mengubah tampilan website, membuat input-output, membuat kalkulator, membuat aplikasi halaman login di Visual Basic 6.0, dan masih banyak lagi. Untuk informasi lebih lengkapnya, Anda dapat melihatnya di sini.

Membuat website tentu memiliki banyak tantangan, mulai dari ambisi yang tinggi namun mudah menyerah, mengikuti tutorial namun menghasilkan output yang berbeda, menemui kesalahan yang sulit dimengerti, membeli domain namun melupakan SSL :) serta kesulitan dalam menyesuaikan tampilan website di desktop, tablet, dan smartphone. FYI, saya menghabiskan satu minggu untuk membuat file index.html dan mengalami tiga kali perubahan. Saya mencoba menggunakan Bootstrap dan Tailwind CSS, namun pada akhirnya kembali ke opsi pertama. Tapi banyak pelajaran yang bisa diambil dari hal ini.
