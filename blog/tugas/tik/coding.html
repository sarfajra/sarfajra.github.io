Apa Itu Coding??

Coding adalah kegiatan yang dilakukan progammer untuk bisa komunikasi dengan komputer. Biasanya coding di gunakan untuk membuat sebuah website atau software.

Menurut saya coding itu seru, bisa bikin hello world saja sudah merasa bisa jadi bjorka. Berhasil bikin hello world itu sama kaya bisa menyelesaikan soal matematika.

Saya mengisi waktu liburan saya dengan belajar coding. Saya belajar html, php, dan css. Untuk text codingannya saya menggunakan notepad++ dan untuk mengaktifkan codingannya saya menggunakan xampp.

Saya sempat mencari-cari website untuk belajar coding. Kemudian menemukan website brightchamps yang kebetulan free trial jadi saya mencoba untuk daftar dan mengikuti kegiatannya.

Sistem belajarnya menggunakan zoom meeting kita share screen supaya kakanya bisa memantau dan membimbing dengan leluasa. Kita belajar phyton di website replit.com
